# 🚗 Road Review & Route Insight System

## 📌 Objective
A web application that helps users evaluate road conditions using crowd-sourced reviews to choose better routes.

## 🛠 Tech Stack
- PostgreSQL
- Express.js
- React.js
- Node.js

## 🚀 Features
- View road routes on map
- Add reviews (potholes, traffic, smoothness)
- Rate road quality
- View aggregated ratings

## 📊 Status
🚧 In Progress – Backend and database design in progress

## 💡 Future Enhancements
- Route suggestions based on road quality
- Vehicle-based route optimization
